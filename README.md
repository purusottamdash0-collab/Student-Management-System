# 📚 Student Management System Using Python

## 📌 Project Overview

The **Student Management System** is a Python-based application that helps manage student records efficiently. It provides a simple command-line interface (CLI) to perform basic CRUD (Create, Read, Update, Delete) operations on student data.

Student information is stored in a **JSON file**, making the application lightweight, portable, and easy to use without requiring a database.

---

## 🎯 Objectives

- Manage student records efficiently.
- Store student information using JSON.
- Perform CRUD operations.
- Learn file handling in Python.
- Understand data persistence using JSON.

---

## 🚀 Features

- ➕ Add New Student
- 📋 View All Students
- 🔍 Search Student by Roll Number
- ✏️ Update Student Details
- ❌ Delete Student Record
- 💾 Automatic Data Saving
- 📂 JSON-Based Storage

---

## 📂 Student Information

Each student record contains:

- Roll Number
- Name
- Age
- Course

Example:

```json
{
    "roll": "101",
    "name": "John Doe",
    "age": "20",
    "course": "Computer Science"
}
```

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/your-username/Student-Management-System.git
```

Move into the project folder

```bash
cd Student-Management-System
```

Run the notebook

```bash
jupyter notebook
```

Or open it directly in **Google Colab**.

---

## ▶️ How to Use

Run the notebook and choose the desired operation from the menu.

Available options include:

1. Add Student
2. View Students
3. Search Student
4. Update Student
5. Delete Student
6. Exit

All records are automatically saved to the **students.json** file.
